#Jugs for Three
Write a program that receives a number on the input.
If the number is a multiple of 3, it prints "Jugs". 
Otherwise, it prints the number.

#input1 
3 
#input2 
112

#code
num = int (input())

if num%3 == 0 :
  
print ("Jugs")

else :
  
print (num)

#output1
Jugs
#output2
112
